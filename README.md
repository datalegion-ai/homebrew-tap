# Homebrew Tap for Data Legion CLI

Official [Homebrew](https://brew.sh) tap for [Data Legion](https://www.datalegion.ai) CLI tools.

## Installation

```bash
brew install datalegion-ai/tap/datalegion-cli
```

## Upgrade

```bash
brew upgrade datalegion-ai/tap/datalegion-cli
```

## Uninstall

```bash
brew uninstall datalegion-ai/tap/datalegion-cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `datalegion-cli` | CLI for the Data Legion API — agent-friendly, fully async |
